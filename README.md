# Quốc Âm Tân Tự (國音新字)

Dự án lấy cảm hứng từ tài liệu lịch sử Quốc Âm Tân Tự, nhằm số hoá, bảo tồn và hiện đại hoá hệ chữ viết cải tiến tiếng Việt.  
Mục tiêu là cung cấp công cụ kỹ thuật để nghiên cứu, học tập và ứng dụng thực tế.

---

## Tính năng

- Mapping chữ Quốc ngữ -> Quốc Âm Tân Tự
- Bộ gõ (IME / Rime schema)
- Demo trên web
- Dữ liệu YAML/JSON dễ mở rộng

---


## Cài đặt

### Clone
```bat
git clone https://github.com/hungtq1016/quocamtantu
cd quocamtantu
```

### Chạy web
```bat
npm install  
npm run dev
```

Xem chi tiết tại: [Đây](./src/build_font_from_png/README.md)

## Mục tiêu dài hạn

- Chuẩn hoá bảng mã
- Tạo font Unicode hoàn chỉnh
- Đưa bảng mã lên bảng Unicode quốc tế
- Bộ gõ đa nền tảng
- OCR tài liệu cổ
- Thư viện học tập cho giáo dục
